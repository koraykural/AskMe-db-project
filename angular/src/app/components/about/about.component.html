What is AskMe?

AskMe is a social media where people can ask any kind of question and get answers.

For building up a better community, members vote questions and owner of the voted question earns or loses askpoints. These askpoints are later needed for asking more questions.

Also, for higher askpoint costs, one can ask a question anonymously.

Details About AskMe?

This web app is a term project for the database course given at ITU.

For database, postgreSQL is used.

For backend, python - flask is used.

For frontend, angular and tailwindCSS is used.
